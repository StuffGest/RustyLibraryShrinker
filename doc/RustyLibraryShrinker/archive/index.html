Gestion de l’extraction des archives de bandes dessinées.

Ce module centralise les mécanismes d’extraction pour les différents formats de fichiers supportés par l’application :

  • Archives compressées (CBZ, CBR) via zip_rar
  • Documents PDF via pdf
  • Livres numériques EPUB via epub

Il fournit une interface unifiée pour décompresser ces fichiers dans des répertoires temporaires avant le traitement des images.
